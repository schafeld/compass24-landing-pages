# Statistics (or key metrics) section

## Feature

A full-width stats or key metrics banner-section to
display Compass24 key metrics for use as promotional topics:

- Gründungsjahr 1979
- Über 42.000 Artikel
- Katalog mit über 400 Seiten
- 5.000 Pakete täglich im Versand
- 11 Länder weltweit

(Source for metrics an reference for design is https://www.compass24.de/ueber-uns)

## Layout and design

- Use fonts and color palette as [Compass](www.compass24.de)
- Blue gradient background
- White font
- One line in large screens (big desktop) reflowing to certical stack on mobile (small screens)
- Wrap css in very specific wrapper class like .compass24-key-metrics-component to avoid side effects when code is inserted into website via CMS
